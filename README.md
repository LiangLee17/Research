# Research

本仓库用于记录个人科研学习中的相关内容。



## 随机数生成

展示了伪随机数产生和发展的历史流程，并说明了可能存在的一些问题。

用算法生成的随机数实际上都是伪随机数，真正的随机数要靠物理现象生成。

现在的随机数生成的主要方法是**梅森旋转**，它在极高维度才能看出其非随机性。



## AFEPack

（Adaptive Finite Element Package）

自适应有限元计算软件包，是北京大学李若教授开发和长期维护，面向数值计算科研人员的计算软件包。
